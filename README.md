# Model Details

This work is part of the **<a href='https://finsurvival.github.io/'>FinSurvival Challenge at ICAIF'25</a>**.  

We experimented with a range of survival models including:

- **Cox Proportional Hazards (Cox PH)**
- **Weibull Accelerated Failure Time (AFT)**
- **DeepSurv**
- **XGBoost** – achieved the **best concordance index (0.8472)**
- **LightGBM**
